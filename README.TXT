# Recipe Meal Planner

## Description

Recipe Meal Planner is a web application that allows users to search for recipes based on various criteria such as ingredients, dietary restrictions, and cooking time. Users can find detailed information about each recipe, including ingredients, instructions, nutritional information, and user ratings.

## Features

- **Recipe Search:** Users can search for recipes using keywords, ingredients, or dietary preferences.
- **Detailed Recipe Information:** Each recipe displays detailed information, including ingredients, instructions, nutritional information, and user ratings.
- **Filtering and Sorting:** Users can filter and sort search results based on various criteria such as dietary restrictions, cooking time, and popularity.
- **Save and Share Recipes:** Users can save their favorite recipes and share them with others via social media or email.
- **Meal Planning:** Users can plan their meals by saving recipes to a meal planner and organizing them by day or meal type.

## Technologies Used

- **React:** Frontend framework for building user interfaces.
- **Vite:** Fast, modern build tool for frontend development.
- **Spoonacular API:** External data source for retrieving recipe information.
- **CSS:** Styling language for customizing the appearance of the application.
- **GitHub Pages:** Hosting platform for deploying the application.

## Usage

To run the application locally, follow these steps:

1. Clone this repository to your local machine.
2. Install dependencies using `npm install`.
3. Start the development server using `npm run dev`.
4. Open [http://localhost:3000](http://localhost:3000) in your web browser to view the application.
